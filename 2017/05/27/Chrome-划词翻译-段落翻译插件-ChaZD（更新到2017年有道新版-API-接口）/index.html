Chrome 划词翻译 & 段落翻译插件 ChaZD（更新到2017年有道新版 API 接口）

由于 Mac 版的有道词典 App 在快捷键呼出界面总是会崩溃（后找到规律：只要切换在段落翻译界面被呼出就会崩溃），给产品提了反馈也一直没有得到更新。今天想到了去找一款好用的 Chrome 扩展顶替，在知乎的一个回答让我找到了 ChaZD。

直接在 Chrome 应用市场安装后，一直提示翻译中，貌似是不可用的，但根据推荐者的一翻推荐，心想应该是个不错的插件，于是追踪到了 ChaZD 的 github，看看 issue 以及看看有没有更新的版本。在 issue 中找到了同路人，原来是 ChaZD 是用的有道 API，说是每个 API Key 有每日 1000 次的查询限制（作者的 Key 肯定不够用啊），立即去了有道开放平台申请了自己的 Key。

结果，有道的 API 在今年已经更新了，老板的 API 已停止申请，且即将停止提供服务，顿时眼前一黑，开始寻觅其它可用的扩展。

但是，没有找到心仪的，不是不能自定义快捷键，就是体验不够轻巧，又或是无法翻译短语和段落。哎，只能自己撸了，打开 ChaZD 的源码，按照有道新版的 API 撸了一份可用的版本。

Github：terranc/ChaZD

需要的朋友需要自行 Grunt 编译后打包使用。在 utility.js 中修改成自己的 API key + 密钥。

不知道如何打包.crx的自行 Google 吧。
